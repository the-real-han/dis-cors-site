DisCORS

Relax securityat the right place

DisCORS is a Chrome extension that allows user to disable common HTTP security headers, and the user can choose to apply it to all HTTP responses or only for responses from a specific site or list of sites. Work with your sites, and still browsing the internet safely with no compromises.

Get DisCORS

Under The Hood

Precise Targeting

There are plenty of extensions that allows user to bypass CORS restriction, but most of them will apply it for ALL responses, which is sometimes not what user actually wanted. With DisCORS, you can chose the precise targets for bypassing CORS restrictions, and it won't tamper with any other sites. - ONLY your targets are impacted - NO security compromises while browsing - Wildcard and subdomains can be used Get DisCORS now and develop with peace of mind!

Flexible Bypassing

Depending on your target, sometimes just dropping/modifying CORS headers are sufficient to bypass CORS restrictions, but other times, you may need a little more control. And DisCORS has the options for other security headers that can help in those special cases: - X-Frame-Options - Content-Security-Policy - Permissions-Policy Learn more about common HTTP security headers!
